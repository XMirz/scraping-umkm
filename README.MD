## Requirement

- NodeJS installed

## How to

- Clone this repo

```
git clone https://github.com/XMirz/scraping-umkm.git
```

- Run command to install dependencies

```
npm install
```

- Change start variable with your starting point

```
let start = 8001; // Start data
```

- Run with this command

```
node index.js
```

- Tunggu beberapa tahun
